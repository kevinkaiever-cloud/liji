# 🌀 量子粒子模拟器 | Quantum Particle Simulator

一个基于 Canvas 2D 的交互式粒子物理模拟网站，结合现代前端动画工作流的最佳实践。

## ✨ 特性

### 🎨 视觉效果
- **实时粒子物理模拟** - 基于 Verlet 积分的物理引擎
- **多彩粒子系统** - 支持彩虹、能量、单色等多种颜色模式
- **动态轨迹效果** - 可调节的粒子轨迹和连线效果
- **交互式控制** - 鼠标/触摸交互添加和影响粒子
- **响应式设计** - 适配各种屏幕尺寸和设备

### ⚙️ 交互功能
- **实时参数调节** - 粒子数量、速度、引力、排斥力等
- **实验模式** - 预设的物理场景（漩涡场、粒子爆炸等）
- **性能监控** - 实时显示 FPS、粒子数量、系统状态
- **主题切换** - 深色/浅色主题支持
- **全屏模式** - 沉浸式体验

### 🛠️ 技术栈
- **HTML5 Canvas 2D** - 高性能图形渲染
- **现代 JavaScript (ES6+)** - 模块化代码结构
- **CSS3 动画** - 流畅的 UI 过渡效果
- **响应式设计** - Flexbox + Grid 布局
- **构建工具链** - 自动化构建和优化

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 部署到 GitHub Pages
```bash
npm run deploy
```

## 🎮 使用指南

### 基本操作
1. **点击画布** - 在点击位置添加新粒子
2. **拖动鼠标** - 创建粒子流
3. **鼠标滚轮** - 调整视图缩放
4. **空格键** - 暂停/继续模拟

### 控制面板
- **粒子控制** - 调节粒子数量、速度、物理参数
- **视觉效果** - 切换颜色模式、调整轨迹效果
- **模拟控制** - 开始/暂停/重置模拟
- **实验模式** - 尝试不同的物理场景预设

### 性能优化
- **GPU 加速渲染** - 利用硬件加速
- **动态粒子管理** - 自动优化性能
- **内存监控** - 实时显示资源使用情况
